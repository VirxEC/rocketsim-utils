# ball_sim_cpp CMake Example

This example builds the Rust static library via Cargo, then compiles a C++ executable that uses the CXX-generated bindings.

`ball_sim.h` is generated by cxxbridge's command line tool:

```
cargo install cxxbridge-cmd
cxxbridge ball_sim_cpp/src/lib.rs --header > ball_sim_cpp/make_example/ball_sim.h
```

## Prerequisites

- Rust toolchain (`cargo`, `rustc`)
- CMake 3.16+
- A C++14-compatible compiler

## Build

From the repo root (defaults to Debug):

```
mkdir -p ball_sim_cpp/make_example/build
cmake -S ball_sim_cpp/make_example -B ball_sim_cpp/make_example/build
cmake --build ball_sim_cpp/make_example/build
```

To build Release (and use Rust `--release`):

```
mkdir -p ball_sim_cpp/make_example/build
cmake -S ball_sim_cpp/make_example -B ball_sim_cpp/make_example/build -DCMAKE_BUILD_TYPE=Release
cmake --build ball_sim_cpp/make_example/build
```

## Run

The simulation expects `./collision_meshes/` to be available relative to the working directory. Run from the repo root:

```
./ball_sim_cpp/make_example/build/ball_sim_example
```
